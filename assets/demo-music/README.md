# Cranamp demo music

This directory is reserved for built-in demo tracks for Cranamp.

The demo tracks exist only to make the app useful immediately after clone or install, especially for testing the player UI, playlist behavior, spectrum/visualizer, seeking, and metadata display.

These assets are generated demo assets, not commercial music. Do not add copyrighted songs, online rips, game soundtrack files, personal-library files, or third-party samples to this folder.

Users can remove these files or replace them with their own local music. Keep replacement music outside this folder unless it is safe to redistribute as part of Cranamp.

No generated binary audio is committed by default. Generate, review, and listen locally before committing any selected `.mp3` files.
